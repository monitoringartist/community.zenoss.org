Dell SNMP Event Transforms

VERSION 3 
Created on: Nov 25, 2009 3:06 PM by jbaird - Last Modified:  Aug 17, 2011 2:55 AM by Nick Yeates

Submitted by:
Josh Baird

Description:

This ZenPack provides the /Events/HW/Dell class which contains over 60 event transforms for events/SNMP traps generated by Dell OpenManage (OMSA).

 

Usage

Event Parsing

Upon installing the ZenPack a new device class named /Events/HW/Dell is created. This class contains over 60 pre-defined SNMP event transforms that turn traps generated by OMSA into a more readable format.

For Example:

Upon a memory failure, the host may send ZenOSS a trap such as alertMemoryDeviceFailure. The pre-defined transforms included in this ZenPack will transform that into Dell OpenManage: Memory Device Failure. The event will also contain other relevant information such as the DIMM location for a Memory related event. See the screenshots below for additional details.

MIBS

This ZenPack inlcudes the necessary Dell OMSA MIBs.

  • Zenoss Version: 2.3, 3.0+
  • ZenPack Dependencies:
  • External Dependencies: Dell OpenManage Server Administrator (OMSA) installed on the host that you are monitoring
  • Installation: Create the /Mibs/Dell organizer
