Update to David Nicklay's Group Resources Report ZenPack

VERSION 3 
Created on: Jun 11, 2010 3:09 PM by jcurry - Last Modified:  Aug 18, 2011 12:28 AM by Nick Yeates

Note: Please see parent page for newest content: Group Resources Report

 

I have augmented this ZenPack to also report on Systems and Locations, in addition to Groups.  I have also added an "export all" button so the data can be exported to CSV files.

 

12th Nov 2010 - Have now updated so that it works with Zenoss 3 (tested on 3.0.3).  This is the 1.4 version.  I will post below.  If you want to install the 1.4 version on a Zenoss 2.5, rename the egg file from ZenPacks.community.groupreporting-1.4-py2.6.egg to ZenPacks.community.groupreporting-1.4.egg and you should be fine.

 

Please test it out and provide feedback.

Cheers,

Jane

Note: binary attachments ending in .zip will need to be unzipped before use.
