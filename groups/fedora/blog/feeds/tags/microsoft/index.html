It&rsquo;s just that it involves a learning curve you&rsquo;d be happy to do without. </li><li>The Microsoft Project Desktop application is like an invisible fortress: the team doesn&rsquo;t see the &ldquo;big picture&#8221;; they only see assigned tasks.
