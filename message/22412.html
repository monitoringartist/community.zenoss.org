    New ZenPacks in the Community
    Community member Blake 'webpass' Drager has generously provided his
    fping ZenPack. This ZenPack contains a template that should be
    valuable for anyone running a network where latency is crucial.
    It includes 4 graphs:
    * An "overall" graph representing latency and loss
    * A basic packet loss graph
    * A graph showing min and max deviation from the average response
    time
    * A graph showing number of pings sent, received, and lost

    fping ZenPack: http://www.zenoss.com/community/projects/zenpacks/fping

    For this and other ZenPacks, check out the ZenPacks site (http://www.zenoss.com/community/projects/zenpacks/
    ).

    Thanks Blake!

    New ZenPacks in the Community
    Michael C. Hoffman has written a new VMware Virtual Machines ZenPack
    for the Zenoss Community. This ZenPack is completely different than
    the current VMware ESX ZenPack. The existing ESX ZenPack grabs the ESX
    file system volumes. This actually creates a new tab with all of the
    hosted virtual machines running on the ESX server. It then plots the
    CPU and Memory usage for each of those virtual machines.

    VMware Virtual Machines: http://www.zenoss.com/community/projects/zenpacks/vmware-virtualmachines

    New ZenPacks in the Community
    Zenoss Community member Sonny Stormes has updated the NetApp Volume
    ZenPack to address the reversed data under the OS tab. With this
    install, the ZenPack was exported from a stack installation, which
    puts all of its data under /usr/local/zenoss. You will want to verify
    that the check_netapp script is available in the libexec directory for
    the ZenPack to work.

    NetApp Volume ZenPack: http://www.zenoss.com/community/projects/zenpacks/netappvolume

    Many thanks Sonny and everyone else who has worked on this ZenPack!

    New ZenPacks in the Community
    The VMwareESX ZenPack has been updated. Based on the discussions in
    the forums regarding the "-k" flag in the VmwareEsxDf.py script, I

    Community ZenPack Repository: trac-zenpacks/wiki/ZenPacksvmwareVMwareEsx

    New ZenPacks in the Community
    Zenoss Community member Michael C. Hoffman has generously provided the
    new HP ProLiant MIBs ZenPack. This ZenPack consists of all the
    publicly available (and loadable into Zenoss) MIBs for HP ProLiant
    servers. The MIBs may be used for translating SNMP traps. NOTE: There
    are a large number of MIBs included in this ZenPack, so it is advised
    that you install with the command-line rather than the UI so it does
    not time-out.

    HP ProLiant MIBs: http://www.zenoss.com/community/projects/zenpacks/hpproliantmibs

    Community ZenPack Repository page: trac-zenpacks/wiki/ZenPacksHPProLiantMIBs
