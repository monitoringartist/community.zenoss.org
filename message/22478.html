    New ZenPacks in the Community
    Community member Egor Puzanov has provided the new NetWare SNMP
    ZenPack. This is a purely SNMP-based ZenPack for monitoring NetWare.
    It does not require the MRTGEXT.NLM module, unlike the NetWare 6.5
    ZenPack. It currently supports:
    1. Filesystem monitoring (used, free, purgeable space)
    2. Filesystem Utilization Reporting
    3. Support for Novell Cluster Service (virtual interface for second
    IP)
    4. CPU Utilization
    5. Basic Memory Usage monitoring

    NetWare SNMP: http://www.zenoss.com/community/projects/zenpacks/nwmon
    Community ZenPack Repository: trac-zenpacks/wiki/ZenPackscommunityNWMon

    Thanks Egor!

    New ZenPacks in the Community
    Thanks to Egor Puzanov for providing his new RDBMS Monitoring
    ZenPack. This ZenPack provides basic monitoring for databases that
    implement the RDBMS-MIB standard (e.g. PostgreSQL (pgsnmpd) and
    Ingres). It is used with the ZenODBC ZenPack as the basis for several
    other database ZenPacks (MySQL ODBC, MS SQL ODBC and PostgreSQL ODBC).
    It provides the community.snmp.DatabaseMap collector plugin, database
    pages, MIBs and the Database Util Report.

    RDBMS Monitoring: http://www.zenoss.com/community/projects/zenpacks/rdbms
    Community ZenPack Repository: trac-zenpacks/wiki/ZenPackscommunityRDBMS

    Thanks Egor!

    New ZenPacks in the Community
    Zenoss Community member Egor Puzanov has generously provided a new
    ODBC Datasource ZenPack. The ZenODBC ZenPack implements the ODBC
    DataSource and uses the commandParser from Zenoss 2.4. It has an ODBC
    Modeler Plugin class (Python Plugin wrapper) which can be used by
    other ZenPacks. It is used as the basis for the MySQL ODBC, MS SQL
    ODBC and PostgreSQL ODBC) ZenPacks.

    ODBC Datasource: http://www.zenoss.com/community/projects/zenpacks/zenodbc

    New ZenPacks in the Community
    Zenoss Community member Egor Puzanov has provided the new MS SQL ODBC
    Database Monitor ZenPack. This ZenPack provides ODBC monitoring of MS
    SQL databases. There are MsSql80Database (MS SQL 2000) and
    MsSql90Database (MS SQL 2005) templates available for binding and a
    community.odbc.MsSqlDatabaseMap zCollectorPlugin that the device will
    need to use.
    The ZenPack provides the following performance graphs:
    * Database Files Size
    * Database Used Space

    MS SQL ODBC Database Monitor: http://www.zenoss.com/community/projects/zenpacks/mssql-odbc
    Community ZenPack Repository: trac-zenpacks/wiki/ZenPackscommunityMsSQLMon_ODBC
