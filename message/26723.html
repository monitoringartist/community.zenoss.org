    These strange MAC addresses can be cleaned up in the transform section
    of the event class mapping for this particular trap.

    evt.vBrdgTpFdbAddress = ":".join(map(lambda c: "%02x" % ord(c), evt.
    vBrdgTpFdbAddress))
