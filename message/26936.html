    Re: Zenoss not alerting on 'quick' open/close events?

    I ran into the same issue as you - (I know this is an old post).

     

    I resolved it by putting a 60 second sleep on my clear events that I want to show up when they flap up/down...aka on my link up event:

     

    Edit tab ==> Transform Section

     

    There I put

     

    import time
    time.sleep(60)

     

    This looks to have resolved it so I see alerts for events that are 'quick' as you say - the drawback is 60 delay on the clear/back up email which is fine for me since we have to look at it regardless.
