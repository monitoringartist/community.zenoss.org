    SNMPD on Centos
    Hi,

    I just installed netsnmp on my centos box as follows:
    yum -y install snmpd

    to start the deamon I issued
    service snmpd start

    then checked snmpwalk (as user zenoss)
    snmpwalk -Os -c public -v 2c -r3 -t10 -m ALL `uname -n`
