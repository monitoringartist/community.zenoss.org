    Re: New Site Issues Thread

    We migrated all of the old wiki and website content and added URL redirects to the new locations.  All of the old threads were migrated with their content, authors and dates, but URLs between forum posts were broken.  The search is greatly improved, so you should be able to find things easier.  Iif you run into posts referring to previous posts that are no longer there, you can reply to them with the new URL.

     

    If you were using the email bridge with the old site, we've preserved that setting.  If you want to disable email notifications for a forum, select "Stop email notifications" on the right side of the forum.  The emails are no longer from the same email address, so if you were filtering on that you may want to update your filter on the subject containing "[zenoss-users]".  We're working on cleaning up the email notification template to be more minimal.

     

    Thanks,

    Matt

    One thing I liked about the old forum system is that it told you at the very top level how many posts there were on a given topic, and how many of them were new.  The new forum doesn't provide that functionality.

    Re: New Site Issues Thread

    I think the new site is excellent.  I love the community driven design and the fact that each user can create their own blog on the site.  It also supports importing content from another blog (such as a wordpress blog).  If you import content from another blog a lot of the spacing gets messed up and URLs get stripped out, so you have to go through and fix it (which isn't all that bad, but mildly annoying, especially if someone were importing hundreds of posts).  The integrated feel of the site is very nice.

     

    I do have a few concerns about the site at the moment.

     

    First off, the top bar where it says "Welcome, <username> (Log out)", the text is white and the background is gray.  It's quite hard on the eyes.  It would be nice if the background were a darker shade of grey, and if the links were a lighter or darker colour.  The contrast between the background and the text isn't what it should be at the moment.

     

    On the main page the "DOWNLOAD - Download Zenoss Installers, RPMs, VMware images, or Source Code." button looks like it has had the top of it cut off.  The other 2 buttons below it are nicely rounded at the top with a black border.  The front page of a site is it's "handshake", and it would be nice if that button matched the other 2, as it's not very aesthetically pleasing at the moment (I'm a sucker for eye-candy).

     

    If you navigate to your profile and click on the "See yourself in the Organization Chart" link at the bottom right, the next screen loads with part of the bottom navigation bar spanning completely across the screen, and with part of it aligned to the far left.  I'm using Firefox, I'm not sure if it does the same thing in IE or not, but it looks like some form of coding issue.

     

    Probably my largest concern is in regards to data that was imported over from the forum and other sources.  The forum post formatting has gotten messed up.  Quotation marks have turned in to something like ".  Code has been imported as some 10 digit alphanumeric character, so the original code is gone (making a lot of posts useless).  I was also browsing through the documentation section and noticed that a lot of the spacing in the Docs is completely messed up.  It's a bit difficult to read a document when it's all mashed together due to all spacing being removed.

     

    Ryan Matte

    Nova Networks
