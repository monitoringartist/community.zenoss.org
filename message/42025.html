    • In Safari 4.0.4, the interface works well all-around.
    • In Chrome 3.0.195.33, the interface works well all-around.

     

    • In Firefox 3.5.5, the Event Console returns a malformed page with no data, but no real error message. To me, this is the one that sounds like the reason this thread got started.
