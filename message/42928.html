    Thanks to Community member Richard Esteve for providing his new WMI MSSQL ZenPack.  This WMI Data Source-based ZenPacks provides a WMIPerfMSSQL template for monitoring Microsoft SQL Server.  The new template is available on the /Devices/Server/Windows device class and provides graphs for many MSSQL counters :
