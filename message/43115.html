    Re: Disk space thresholds not working

    I lucked out, I had an old 2.4.5 VM on my backup server..

     

    For me on a 2.4.5 installation, I could only get things to start freaking out when both thresholds were set to the same name.

    Here is what I have for template filesystem thresholds:

     

    Free Space 90 PercentMinMaxThresholdusedBlocks_usedBlocksWarningTrue
    Free_Space_98_percentMinMaxThresholdusedBlocks_usedBlocksErrorTrue

     

     

    NameFree_Space_98_percent
    Data Points
    Min Value
    Max Value
    Event Class
    Severity
    Escalate Count
    Enabled

     

    The 90% threshold is identical except for criticality, multiplier, and event class. I use /Perf/Filesystem for 90%, warning, and .9
