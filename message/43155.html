    Re: Event Console - Server Connection Error

    This one isn't zenpatchable because it only applies to the trunk, not the 2.5.x branch. However, you can drop the attached handleUnicodeEvents.diff file into your $ZENHOME/Products/ directory and run the following commands as the zenoss user to apply it.

     

    cd $ZENHOME/Products

    patch -p0 handleUnicodeEvents.diff

    zopectl restart

    ok !!

     

    Just edit file $ZENHOME/Zenoss/Products/ZenUtils/json.py

    At line 49 :

     

    replace

     

    return simplejson.dumps(value)

     

    by

     

    return simplejson.dumps(value, encoding='iso-8859-1')

     

    restart Zope and that's it !! Works great!! My event console was broken, I applied this patch and now without purging MySQL database it's working !!!

     

    Thanks Chet !!

     

    (PS : Andrea, the patch in incorect, just use $ZENHOME/Zenoss/Products/ZenUtils and the command should work)
