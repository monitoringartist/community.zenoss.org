    Re: Help with Custom Map Zenpack

    Actually, the easiest way I have found to deal with maps that begin with numbers or have dots or spaces in their names is to use the zendmd find() function:

     

    from ZenPacks.USU.Map import CustomMap
    loc=find("/zport/dmd/Locations/City/FactoryA/ServerRoom/81")
    loc.custom_map = CustomMap.CustomMapObject(loc)
    loc.custom_map._registerCustomMapTab()
    commit()

    Here is a zenpack that attempts to address some of the problems with the original download.

     

    Fixes:

    • Automatically installs PIL if it is not found
    • Automatically registers the custom map portlet
    • Transition effects are disabled
    • Automatically adds the custom map tab to any existing locations
    • Custom map methods quit if they are being applied to the wrong location

     

    Problems: (please help if you can!)

    • Need to automatically add custom map tab and custom_map object to new locations
    • Need a good way to upload background images. Currently, images have to be loaded into zope via a zenpack or some other method.
