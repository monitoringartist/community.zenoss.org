    Re: Data Collector Daemons

    After research, I have summized the following regarding which Zenoss daemons should run on the Primary Server and Data Collector(s).

     

    Zenoss Primary Server and Data Collector Daemon Notes

     

    • User Layer zopectl daemon only runs on the the Zenoss Primary Server.
    • Zenoss Process Layer (zenhub, zenactions and zenjobs) daemons only runs on the Zenoss Primary Server.
    • ZEO back-end object database, zeoctl daemon only run son the Zenoss Primary Server.
    • All other daemons run on both the Zenoss Primary and Data Collector(s).

     

    Can someone confirm the above to be valid. If so, can we document somewhere?

     

    - Ken
