    Re: Move Events to History every hour

    In case anyone is interested I found a way to convert evt.lastTime to epoch time in python.

     

    Here is a test performed in zendmd:

     

    >>> import time
    >>> evt = dmd.ZenEventManager.getEventDetailFromStatusOrHistory('ae3b4cbe-a2f4-40bd-8724-1c796890bc54')
    >>> pattern = '%Y/%m/%d %H:%M:%S'                                              
    >>> epoch = int(time.mktime(time.strptime(evt.lastTime.split('.')[0], pattern)))
    >>> print epoch
    1280339318

     

    Works like a charm.

     

    Cheers.
