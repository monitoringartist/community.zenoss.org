Nagios plugins fail with "Code: 2 - Msg: Misuse of shell builtins" in v2.5.2

Last week I upgraded from v2.4.5 to v2.5.2. Since then, all the Nagios-style plugins I have installed through custom templates are getting the error "Code: 2 - Msg: Misuse of shell builtins". Apparently Zenoss is interpretting the scripts' exit codes (part of the Nagios plugin standard) as errors instead of information.

 

I did see the message in the Release Notes about changing the plugin type from "Auto" to "Nagios", but that hasn't resolved it.

 

I am using quite a few different scripts on different servers, so I really hope I don't need to re-write all the scripts. The Nagios format is still supported, righ?
