    I wasn't able to find out how to "clear the catalogue", but I did find out to do the following and it appears to have worked.

     

    login as zenoss user

    run zendmd

    reindex()

    commit()

    exit (ctrl+d)

    /etc/init.d/zenoss-stack restart
