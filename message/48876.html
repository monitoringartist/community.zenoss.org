New Amazon Web Services ZenPack

New in Zenoss 2.5, the Amazon Web Services ZenPack allows you to monitor Amazon Elastic Compute Cloud (Amazon EC2) server instances. It collects information for these objects monitored through Amazon's CloudWatch APIs.

 

This ZenPack is documented in the Zenoss 2.5 Extended Monitoring Guide and source is available at http://dev.zenoss.org/trac/browser/trunk/zenpacks

    Re: New Amazon Web Services ZenPack

    The links on the Google map are populated by the interfaces on your devices.  So if a device has been modeled and has an "eth0" and a network it's attached to, if you assign it to a different location than other device that are attached to that network, they'll be connected by a line on the Google map.
