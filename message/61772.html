    Re: ayuda con Edit Data point

    You might want to read our basic administration guide at http://docs.zenoss.org for a few hours.

     

    Look into thresholds. You can set your own threshold on any datasource (graph). If the data point goes over or under the threshold, an alert is sent.

     

    If you want to know that something is under a threshold for a specified period of time, this is much harder and will require quite a bit of learning. One method: You would have to use an event transform like the one here: message/49662#49662
