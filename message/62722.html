    Re: New Amazon Web Services ZenPack

    Common issues with EC2 are:

    • a bug in our software: update to latest version for major improvement if you have or have had anywhere over 12 instances.
    • Amazon throttles how many api requests you can request and get back answers for - talk to them if you are doing anything on scale
    • Assure that https port and DNS to amazons servers have a clear path through your network and border devices. Some companies are known to block this stuff.
