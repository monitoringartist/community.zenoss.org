    Re: Zenpack installation via Webinterface works?

    I just downloaded the 2.6 file your link and it installed with no issues. This is the link I used: http://community.zenoss.org/servlet/JiveServlet/download/3386-26-3224/ZenPacks.Nova.Windows.SNMPPerfMonitor-1.6-py2.6.egg.zip

     

    Once downloaded unzip and was indeed left with a .egg file. So if you didnt get a .egg double check what file you actually downloaded.

    From there it install as normal (As outlined in the Admin Guide).

     

    Doing a zenpack list while logged in as the zenoss user also shows it as installed.

    [zenoss@localhost ~]$ zenpack --list | grep Nova

    ZenPacks.Nova.Windows.SNMPPerfMonitor (/opt/zenoss/ZenPacks/ZenPacks.Nova.Windows.SNMPPerfMonitor-1.6-py2.6.egg)

    As far as log files, you probably want to check $ZENHOME/logs/event.log or $ZENHOME/logs/zenhub.log
