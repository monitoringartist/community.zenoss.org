    Re: Little help getting started?

    Well, there really isn't a configuration for recieving SNMP Traps - you just need the device in Zenoss somehow - probably a ping only device if you're only using traps... Turn off any other SNMP Trapsinks on the zenoss server so zentrap can grab the traps, and it pretty much goes then. You set the trapsink on the monitored devices to  be the zenoss  server.

     

    Then you're into the admin guide and event mappings and maybe transforms to make the trap generated events in Zenoss be what you want.
