Question on Managing Transforms

I have some general questions on how are folks are handling the managemement of transforms?

 

We dont do any alerting (yet), just trending out of Zenoss, but we are starting to think about it, and as I've been playing a little bit more with transforms.

 

1) Are you keeping your transforms in Source Control?

1a) If so how do you organize them? By event class? Flat directory?

 

2) How would you recover/migrate transforms if you were creating a new install and not upgrading in place? If I understand it right you can apply the transform to an event class, but you have to add the event class to a ZenPack not the transform itself?

 

3) If you wanted to know all the event classes that someone (or a team) has applied transform to, where/how would you do it? DMD Script?
