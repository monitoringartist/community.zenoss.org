    Re: Can't access MIBs page after upgrade to 3.2

    bojan:

     

    You could try rebuilding the global catalog. I don't know if it'll work, but it's worth a shot.

     

    Please backup your ZODB before doing this. (Do a zeopack -h localhost -p8100 and copy the Data.fs.old somewhere safe)

     

    As user zenoss, `python $ZENHOME/Products/ZenUtils/zencatalog.py --createcatalog --forceindex`

     

    Once done, restart zenoss and check if you can view the MIB page.

     

    Thanks,
