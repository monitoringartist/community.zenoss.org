Distributed monitor/collector not work:zenhub username/password combination is incorrect

hi all,

 

 

I am now working with zenoss in distributed environment , all master and collectors running on Centos with zenoss version 4.2.2 .

 

After config a collector ,following instructions in this forums, I restart zenoss service ,

 

Then I tail the zenoss status log file , got below errors :

 

 

-     -     -     -     -     -     -     -     -     -     Part one

 

INFO                  zen.zenstatus:         Connecting to 192.168.XXX.XXX:8789

WARNING         zen.pbclientfactory: Cancelling attempts to connect

CRITICAL          zen.pbclientfactory: zenhub username/password combination is incorrect!

ERROR             zen.zenstatus:         Timeout connecting to zenhub: is it running?

 

-     -     -     -     -     -     -     -     -     -     Part two

INFO                zen.maintenance:        Performing periodic maintenance

WARNING       zen.zenstatus:              No service named 'EventService': ZenHub may be disconnected

ERROR           zen.maintenance:         Maintenance failed. Message from hub: ZenHub is down

ERROR           zen.collector.config:      Task configLoader configure failed: ZenHub is down

 

 

I try a few times ,but still got error messages as above .

 

Could any one give me hit for solve this problem?

 

thanks in advance !

    In each of the performance daemon conf files on the collector uncomment the hub user and passwd entries and set them to what's in your $ZENHOME/etc/hubpasswd. Once done restart everything.

    Re: Distributed monitor/collector not work:zenhub username/password combination is incorrect

    To summarize Shane Sir's Answer :

     

    1: On Zenoss Master Server

    2: Run :  cat $ZENHOME/etc/hubpasswd

    3: On Zenoss Collector

    4: Go to /opt/zenoss/etc : Run cd /opt/zenoss/etc

    5:  Below are performace daemons

     

      

    1: zenmodeler

    2: zenwin

    3: ZenWinPerf

    4: zenperfsnmp

    5: zenping

    6: zenstatus

    7: zenjmx

    8: zencommand

    9: zeneventlog

    10: zenprocess

    11:zentrap

     

    6: Vi each of above daemon conf file . Ex : vi zenmodeler.conf

    7: Uncomment below lines .

                          hubusername admin

                           hubpassword ***************

     

    8: Fill above **** with the content seen at step 2 .
