Free Zenoss Event Management seminar - May 15 2013

Zenoss is hosting a second run of my Zenoss Event Management seminar which discusses the architecture section of the Zenoss Event Management paper that you can find here - http://www.skills-1st.co.uk/papers/jane/zenoss4-events/ .

 

This is a FREE 2-hour webinar starting at 10:00 EST, 15:00 British Summer Time, 16:00 for most of Europe, late in Asia Pacific ( if there is sufficient demand we could schedule another session more appropriate to that region).

 

The agenda will be:

  • Quick look at Zenoss 3 events architecture
  • Zenoss 4 events architecture
    • subsystems
    • daemons
    • databases
  • 10 minute comfort break
  • Event life cycle
    • Event generation
    • Device context
    • Event class mapping / event context / event transforms
    • Database insertion and duplicates
    • Resolution and ageing
  • Questions

 

Please register for the event here - https://zenoss.webex.com/zenoss/j.php?ED=165881562&RG=1&UID=1139716977&RT=MiMxMQ%3D%3D .

 

If you find that a useful introduction, consider registering for the 3-day (chargeable) workshop that explores all the details of the paper in-depth.  Details at message/72840#72840 .

 

Cheers,

Jane
