    Re: Enhancing users and roles in Zenoss Core

    I have updated this beta ZenPack for Zenoss 4.x and fixed the issue where you manipulate roles and it complains that you are providing 3 arguments when it only wants two.  I have also changed the logging so that all logging is now to $ZENHOME/log, not hard-coded.

     

    I am still seeing issues with the devices I can and cannot see - not sure whether this is a local issue.  I would much appreciate other folk trying this and providing feedback.

     

    Note that the changes to fix the arguments bug makes the new ZenPack 1.1 version incompatible with Zenoss 3.x so:

    Use 1.0.1 with Zenoss 3.x   https://github.com/jcurry/ZenPacks.skills1st.UserRoles/tree/master    (the master branch)

    Use 1.1 with 4.x          https://github.com/jcurry/ZenPacks.skills1st.UserRoles/tree/4.x     (the 4.x branch)

     

    You can use the "Download ZIP" link on the right of the screen to download the development code.

    If you just want the standalone egg file, download the zip file, unpack it somewhere safe and then navigate down the "dist" directory - you should find ZenPacks.skills1st.UserRoles-1.1-py2.7.egg in there.

     

    Also trying to get this updated on the Zenoss wiki at http://wiki.zenoss.org/ZenPack:ZenPacks.skills1st.UserRoles but not got there yet - watch this space!

     

    Cheers,

    Jane
