    Re: BLOCKED by --allow-hosts

    The "BLOCKED by --allow-hosts" is usually caused by the wrong format for a ZenPack.  What version of Zenoss are you on?  Are you trying to install a ZenPack for 3.x on a 4.x system?  There is often (but not always) a clue in the name of a ZenPack - if it has py2.4 in its name, it is compiled for Python 2.4 which was used by Zenoss 2.x; py2.6 implies Python 2.6 and Zenoss 3.x; py2.7 implies Python 2/7 and Zenoss 4.x.

     

    Cheers,

    Jane
