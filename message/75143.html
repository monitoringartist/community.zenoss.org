    5. Oct 31, 2013 1:08 PM (in response to Jeff Creek)
    Re: Adding any new device stays pending

    It is a file handle issue

     

    # ulimit -Hn

    4096

    # ulimit -Sn

    1024

     

    #lsof -u zenosss | wc -l

    3866

     

    Open files was over soft limit and almost to hard limit.  Have to edit /etc/security/limits.conf.  Then restart zenoss.

     

    #vim /etc/security/limits.conf

    zenoss soft nofile 10240

    zenoss hard nofile 10240

     

    #su - zenoss

    $zenoss restart
