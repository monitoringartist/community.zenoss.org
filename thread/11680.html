    Re: Openlayers Map - how to configure?

    doncius:

     

    I use Fedora myself, but not for Zenoss. Fedora changes often and therefore often breaks Zenoss in very cryptic ways. This is likely a case of such. For Zenoss or other critical server utility I would recomend trying CentOS. If you're native to RedHat (since you use Fedora) CentOS will be familiar. CentOS is a free, limited support, package compatible clone of RedHat Enterprise Linux. Maybe migrate to this and try again? http://centos.org/

     

    Also, if your CPU supports long mode (x86-64) you should use it! Zenoss is much faster and has a much higher capacity on a 64-bit machine compared to comparable 32-bit machine. This is a artifact of Python runtimes. Check for long mode support by issuing command as root 'cat /proc/cpuinfo | grep lm' If you see output, this means you have 'lm' CPU flag and therefore support for 64-bit.
