    Re: recommended Linux partitioning for new install??

    This is how I like to setup my Ubuntu server:

     

    1- I use 2 hard drives, and setup them up as a mirrored raid.

    2- If using 9.04, I setup 3 partitions: /, /images, swap

        If using 9.1, I setup 4 partitions: grub 2, /, /images, swap

     

    I install the whole system on the root partition, I use the dedicate swap partition.  Once the Zenoss setup is completed, I will boot off a system rescue CD or flash disk, and clone the root partition into an image on the /images mount point.

     

    This way if one of the hard drive fails, the system keeps running, if I have a catastrophic system/software failure, I can start off the rescue system and re-image my root partition from the stored image.
