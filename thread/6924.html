    RE: New ZenPacks in the Community
    Community member Florian Deckert submitted this ZenPack for monitoring HP ProCurve Switches. This ZenPack provides the following features:
    - Register new /Manufactures/HP/Products/ ...ProCurve... OID
    - Import basic HP mibs
    - Graph CPU usage for HP ProCurve switches and trigger an event if the CPU load if over 50%
    - Check power supplies and fans status and trigger an event if one fails
    - Convert basic traps to events

    The serial number from the modeler is not currently retrieved. The ZenPack page has the source code they wrote so if you have access to an HP ProCurve and want to fix it and add it to the ZenPack.

    HP ProCurve Switches ZenPack: http://www.zenoss.com/community/projects/zenpacks/hpprocurve

    For this and other ZenPacks, check out the ZenPacks site (http://www.zenoss.com/community/projects/zenpacks/).

    Thanks Florian!
